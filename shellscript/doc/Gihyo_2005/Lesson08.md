# Lesson08

## シェル関数

定義

関数の定義方法は次のようになる

```shellscript
function_name() {
    commands
}
```

「`prg0801.sh`」

引数を渡すこともできる。「`prg0802.sh`」

戻り値の指定も可能「`prg0803.sh`」

コマンドに使用する「`&&`と`||`」
`command1 && command2`:１が成功したら２を実行（１が失敗したら２は実行されない）
`command1 || command2`:１が成功したら２を実行しない（１が失敗したら２は実行　）
